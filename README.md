# Java Paint Calculator

## Description

This is a Java console application that calculates the area of a wall and the amount of paint needed based on user input.

The program prompts the user to enter wall height and width, calculates the total square footage, and determines how many gallons of paint are required.

This project was developed as part of coursework at Southern New Hampshire University.
